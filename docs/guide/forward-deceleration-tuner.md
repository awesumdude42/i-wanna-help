# Deceleration Tuner

 The deceleration tuner determines how fast your robot stops when power is cut from the motors. The tuner will run the robot for 30 inches forward and then cut power. Make sure you have enough space to run this test Strafe Deceleration Tuner: The Strafe deceleration tuner determines how fast your robot stops when power is cut from the motors. The tuner will run the robot for 30 inches to the left and then cut power. Make sure you have enough space to run this test

## Works With

- ✅ Mecanum Drive
- ✅ Swerve Drive
